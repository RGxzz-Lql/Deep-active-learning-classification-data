灰色预测的模型简称GM模型，它所需建模信息少、运算方便、建模的精度较高，因而在各种预测领域有着广泛的应用。	3

GM模型在预测中，对历史数据作不同的取舍时，其预测值并不相同，即这种预测结果将是一个预测值的区间，这就给预测人员的取舍带来一定困难。	3
本文把GM模型与人工神经网络模型结合起来，用对历史数据作不同取舍的GM模型进行预测，然后用人工神经网络确定这些不同GM模型的灰色预测原理和模型GM模型是用原始数据生成后建立的微分方程。	3
由于GM模型的解是微分方程的解，是指数曲线，因此要求生成数列是递增的且接近指数曲线。	3
电力系统负荷本身均为正值，经一次累加生成后即变为递增数列。	3
具体来说，在年电量预测中，可用最近多年的年电量来建模；	3
在月电量预测中，可用最近同类型月的电量来建模或用最近的月的电量数列来建模；	3
在日负荷预测中，可把最近同类型日同一小时的负荷或同类型日前一天同一小时和同类型日同一小时的负荷等组成一个序列来建模。	3
由此预测出未来一日同一小时的负荷，经过24次建模可预测出24小时的负荷。	3
以上的各种序列又可取用不同长度的数列建模与预测。	3
如果要用GM模型预测未来几日的负荷或电量，用上述方法预测出一日的负荷或电量后，把此负荷或电量值再加进原始数列中，相应地去掉数列开头的一个数据建模，再进行未来第二日的预测。	3
依此类推，预测出需要天数的负荷或电量。	3
这种方法称为等维灰数递补模型，它可实现较长时间的预测，但愈往后其灰度愈大，即其准确度愈小。	3
GM模型与人工神经网络的组合GM模型允许用不同的原始数列来建模，可得出不同的模型。	3
具体来说，取原始数列x的不同数据建立的GM模型，因其参数a，u取值不同，所得出的预测值就不同，这时预测的结果是一个区间。	3
实际上，往往需要一个确定的值，由此进行系统的规划与运行安排。	3
这时就应在不同GM模型中选择出一个预测最准确的模型来，但是，不同阶段最好的GM模型并不唯一，这给选择模型造成了较大的困难。	3
此外，由于影响负荷变化的因素较多，同一种模型预测风险性也较大。	3
上述组合预测方法，只对权重进行线性组合，要求参加组合预测方法的误差能保持稳定，但电力系统负荷预测结果的误差往往是非均匀性的，因而这种组合校正就存在不足。	3
选择与预测量最近的n个已知值作为样本，即把最近的n个已知值作为输出，采用不同的GM模型对这n个已知值分别进行预测，其预测值即为神经网络的输入，由此对神经网络进行训练。	3
学习方法采用改进的BP算法，对于训练好的神经网络，当输入端为各种GM模型的预测值时，其输出即为用神经网络组合后的预测值。	3
